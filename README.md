# SISTEMA INTELIGENTE DE GERENCIAMENTO DE TRÁFEGO BASEADO EM APRENDIZADO DE MÁQUINA 

## Introdução
### Objetivo do Sistema
O sistema inteligente de gerenciamento de tráfego baseado em aprendizado de máquina consiste basicamente no controle do tráfego em cidades, gerenciando o fluxo de sinais de trânsito em prol da performance. O projeto está aplicado num cruzamento simples de duas vias, no qual sua função é analisar a quantidade de carros de ambos os lados e decidir, através do aprendizado de máquina, qual sinal e por quanto tempo devemos liberar a passagem dos carros. O software será implementado num embarcado robusto com capacidade de processar um grande volume de dados, com o objetivo de gerar resultados precisos para o contexto no qual está inserido. Nesse mesmo embarcado, teremos um modelo de aprendizado de máquina aprendendo constantemente sobre o tráfego, em busca do melhor fluxo para o cruzamento.
